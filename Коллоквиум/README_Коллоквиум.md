# Коллоквиум "Совместная работа двух Design Patterns"
### Singleton & Abstract Factory


##### В качестве примера реализации совместной работы двух Design Patterns мной были выбраны паттерны Singleton и Abstact Factory. 
##### С помощью совместной работы двух этих паттернов была реализована конфигурация проекта. В моем конкретном примере конфигурация зависит от выбранной социальной сети: VK, Instagram, Telegram. 
##### Реализация Singleton заключается в создании приватного конструктора. Благодаря созданию статистического метода getInstance(), который не позволяет создавать несколько объектов конфигурации одновременно. Следовательно Settings, реализованные в примере, могут быть только одни. 
##### В данном примере в качестве абстрактной фабрики выступает класс Settings, продуктами фабрики (абстрактными объектами - классы LinkGenerator и PhotoRequirements). В примере реализованы также конкретные объекты - конкретные реализации абстрактных объектов, соответствующие классы: VkLinkGenerator, ... и VkPhotoRequirements, ... .
